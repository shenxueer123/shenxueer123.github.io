  • 如何保持快乐
  • ‘驯良些吧，让别人去卖弄聪明吧’
    • 通篇讨论道德与智慧。在我的理解里，道德是用来保护我多些，因为做不道德的事，我心里承受不来。
  • 几个不易答复的问题
  • 世界上最宽阔的是海洋，比海洋更宽阔的是天空，比天空更宽阔的是人的胸怀。-----维克多.雨果
  • 假使你是一个犯人
    • 刑法适用于犯人，比刑法适用于所犯的罪，还是重要的多。
  • 根据于成见的思想
    • 写的是个啥？木有任何感悟 。
  • 为什么我做不好
    • 站在别人的角度思考问题 ，就很容易做好了啊 。
  • 你是上进的吗？
    • 上进心是一种动力，总是给予人力量 ，握住成功之门的钥匙 。
  • 你容易受欺骗吗？
    • 题不对文翻译的问题
  • 你渴望受关注吗？
    • 当然啊,人活着不就是为了别人对你努力的肯定啊 ，没错的 。
  • 你有一种自卑的感觉吗？
    • 放屁的时候会自卑的，剩下的时间没有的 。
    • 人类最普通的一种错误，便是易于走极端，你以为有些事情可以因乐观和自信促进不少，于是你便以为这是至好的决策，作为你的人生哲学唯一的向导 。
    • 你以为一切的成功 ，都是因热烈的信仰,坚决 ，努力 ，这实在是一种走极端的错误 。
    • 现在的世界不需要人在播音台上催促人奋斗 ，而是需要适当的引导 ，教人如何去尝试 。
    • 高尚的理想，正直的意志 ，要成功的决心
  • 我们是由内分泌腺所支配的吗？
    • 一部分吧 。
  • 你会为梦烦恼吗？
    • 不太做梦哈哈哈哈哈哈
  • 依赖性有医治的办法吗？
    • 这个问题实际还是态度的关系，我们对于这件事的态度如何 。一旦独立后，那些有依赖的人便会去运用自己的脑力了 。
  • “差不多 ”的人究竟是什么毛病
    • 从现代心理学说 ，专心便是精神安适最主要的成分 。
  • 记忆可信吗？
    • 记忆是不可信的 ，总会有疏漏 。因为感受而导致的 。
  • 什么是歇斯底里？
    • 都写的这是啥啊 ，前面的文章还挺有意思的，怎么越往后面越让人觉得水了 ，不想看了 。
