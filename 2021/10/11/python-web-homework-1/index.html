python-web.homework-1

Author: shenwansan Date: October 11, 2021  15:21:40 Category: project

最近在学习python-web，写一下项目的感受吧.
作业1，使用socket的底层网络协议，以一个客户端（如谷歌浏览器）的角度，去写一个爬虫的代码。
这比直接取调用request的库，更加底层，更能深入的了解爬虫技术。
课程代码是从底层一步步开始，逻辑思路无比清晰，学习吧。。。骚年

第一步：主函数的思想，也就是直接写出这个函数的输入和输出。

输入：爬取的网址
输出：网址的内容

def main():
    url = 'http://movie.douban.com/top250'
    r = get(url)
    print(r)

if __name__ == '__main__':
    main()

第二步：对封装的主函数get()进行编写。

对于一个客户端
1.创建连接
2.发送http请求头
3.接受响应 

def get(url):
 protocol, host, port, path = parsed_url(url)

 s = socket_by_protocol(protocol)
 s.connect((host, port))     ## 创建连接

 request = 'GET {} HTTP/1.1\r\nHost: {}\r\nConnection: close\r\n\r\n'.format(path, host)
 encoding = 'utf-8'
 s.send(request.encode(encoding))   ## 发送请求

 response = response_by_socket(s)
 print('get response, ', response)
 r = response.decode(encoding)     ## 接受响应

 status_code, headers, body = parsed_response(r)
 if status_code in [301, 302]:   
     url = headers['Location']
     return get(url)              ## 重定向：如果响应发现301,302，在Location字段，重新获取网页。

 return status_code, headers, body

第三步：对各个部分进行编写。

URL：http://movie.douban.com/top250 进行解析
protocol = http , host = movie.douban.com , port = 80 , path = /top250
